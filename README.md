# AI Opportunity Finder

AI Opportunity Finder is an AI-powered career intelligence platform designed to help Computer Science and IT students, graduates, and aspiring software engineers understand what they need to do next to reach their dream careers.

## What’s included

- Next.js App Router with TypeScript
- Tailwind CSS and modern design system
- Prisma schema for PostgreSQL
- UI scaffolding for landing page and dashboard
- Modular feature architecture ready for AI, GitHub, LinkedIn, CV, and roadmap integrations

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create a `.env.local` file with `DATABASE_URL` and any API keys.
3. Run development server:
   ```bash
   npm run dev
   ```

## Scripts

- `npm run dev` — start local development server
- `npm run build` — build production app
- `npm run start` — run production server
- `npm run lint` — run ESLint
- `npm run format` — format source files with Prettier

## Architecture

- `app/` — Next.js App Router pages and API routes
- `components/` — reusable UI and marketing components
- `lib/` — utility modules and integration helpers
- `prisma/` — database schema and Prisma client

## Vision

This repository is scaffolded as the foundation for a premium AI career platform with personalized recommendations, progress tracking, GitHub and LinkedIn analysis, CV auditing, internship matching, and a polished dashboard experience.
